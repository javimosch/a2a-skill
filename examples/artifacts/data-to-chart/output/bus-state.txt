    1. Think of a real-world dataset to generate — examples:

       - Website response times (ms) over a 24-hour period

       a2a send analyst DATA_START\n<your CSV data>\nDATA_END --from fetcher

    ...

[12:03:00] #2 collector -> analyst
    Your task: You are the data analyst. Wait for the fetcher to send you CSV data.
    
    Steps:
    1. Receive the fetcher's message:
       a2a recv --as analyst --wait 60
    2. Parse the CSV data (timestamp,value format)
    3. Compute these statistics:
       - Number of data points
       - Minimum, maximum, mean, median
       - Trend direction (increasing, decreasing, or stable)
       - Volatility (standard deviation of changes between consecutive points)
    4. Send the analysis + the raw values array to the plotter:
       a2a send plotter ANALYSIS_START\n<your analysis as markdown>\nVALUES:[val1,val2,...]\nANALYSIS_END --from analyst
    
    The VALUES must be a JSON array of floats that the plotter can use.
